SecureScan Pro
Privacy-first iOS document scanner built with SwiftUI, VisionKit, Vision OCR, and CloudKit.

Features
- Document scanning using VisionKit
- On-device OCR using Apple Vision
- User-controlled privacy masking
- Local saved scan history
- Search saved scans
- Saved scan detail screen
- CloudKit iCloud backup and restore
- MVVM architecture

Tech Stack
- Swift
- SwiftUI
- VisionKit
- Vision
- CloudKit
- MVVM
- Local JSON storage

Privacy Approach
SecureScan Pro processes OCR on-device and lets users decide whether sensitive information should be masked before viewing or saving scan text.
